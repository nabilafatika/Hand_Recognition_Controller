Hand Recognition Controller

The Hand Gesture Controller is an innovative project that enables users to interact with their computer using hand gestures. By leveraging computer vision and machine learning technologies, specifically the Mediapipe library for hand tracking, this system recognizes and interprets various hand gestures to execute corresponding commands on the computer.

Find Out More

Let's Get Try it!

Users can control the mouse pointer with their hand movements, providing an intuitive and touch-free way to interact with their computer. Certain gestures can be used to adjust system settings such as volume and brightness, enhancing the functionality beyond basic mouse control. Using the computer's webcam, the system captures and processes video frames in real-time to track hand movements and detect gestures.

Let's Go Start!

How to Stop Hand Recognition

To stop or end the hand recognition, you can press Ctrl+C in your terminal. This will turn off the webcam and stop the hand recognition. Termination of the hand recognition can also be done by pressing the Enter key in the webcam window.
